The scripts in this folder simply conduct a spligt half 
spotlight analysis. The data is split by session (odd and even). 

MSCSplitHalfDeconn.m: Split-half version of MakeRestDconnsMSC_Sim.m

createSptlcorr_MSCdconnsDMS.m: Split-half version of createSptlcorr_MSCdconns.m

MakeRestDconnMSC_OE_NetworkTemps.m: Split-half version of MakeRestDconnMSC_NetworkTemps.m

createSptlcorr_MSCdconns_OE_NetCorr.m: Split-half version of createSptlcorr_MSCdconns_NetCorr.m

OddEvenSplitSpotlight.m: This script is based on SpotlightGroupNetworksWashU.m and conducts a split-half spotlight analysis.

OddEvenTableMaker.m: This script makes a table based on output from OddEvenSplitSpotlight.m. Data in the table can used to compare the odd and even solutions.
