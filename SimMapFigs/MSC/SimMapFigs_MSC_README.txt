This folder includes scripts used to plot the similarity to the group average profile for the Power PC hub locations in the MSC subjects and calculate associated summary stats

AveSpatCorrMSCStats.m: Gets summary stats for output of AveSpatCorrToGroupForPowerTop10MSC.m

AveSpatCorrToGroupForPowerTop10MSC.m: This script gets the average similarity to the group across the vertices of each Power PC hub location for each of the 9 MSC subjects

MSCHubFigMaker.m: Makes a color matrix figure based on output of AveSpatCorrToGroupForPowerTop10MSC.m
