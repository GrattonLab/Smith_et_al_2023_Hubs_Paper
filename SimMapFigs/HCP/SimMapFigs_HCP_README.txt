This folder includes scripts used to plot the similarity to the group average profile for the Power PC hub locations in the HCP subjects (752 subjects).


AveSpatCorrToGroupForPowerTop10HCP.m: This script calculates the similarity of the HCP subjects to the group average connectivity profile for the 10 Power PC hubs.

HCPViolinPlotMaker.m: This script makes violin plots of the similarity of the HCP subjects to the group average connectivity profile for the 10 Power PC hubs.
